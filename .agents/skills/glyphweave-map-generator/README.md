# GlyphWeave Map Generator

Agent Skill for generating GlyphWeave `.gemap` tilemaps from natural language descriptions.

## Installation

This skill is installed locally at `~/.agents/skills/glyphweave-map-generator/`. Pi auto-discovers it.

## Usage

Ask the agent to generate a map, for example:
- "Generate a 3-room dungeon with a treasure vault"
- "设计一个带熔岩河的地牢"
- "Add a prison wing to this map"

## Requirements

- The GlyphWeave Render API running (`pnpm dev` or gh-pages) for map previews
